# Compliance Checklist Templates

## General Compliance Assessment

### Governance
- [ ] Compliance policy documented
- [ ] Compliance officer/team designated
- [ ] Board/executive oversight
- [ ] Regular compliance reviews scheduled
- [ ] Third-party audit program

### Risk Assessment
- [ ] Compliance risks identified
- [ ] Risk register maintained
- [ ] Mitigation plans documented
- [ ] Regular risk reassessment
- [ ] Incident tracking

### Policies & Procedures
- [ ] All required policies documented
- [ ] Policies reviewed annually
- [ ] Version control maintained
- [ ] Employee acknowledgment
- [ ] Exception process defined

### Training
- [ ] Initial compliance training
- [ ] Annual refresher training
- [ ] Role-specific training
- [ ] Training records maintained
- [ ] Effectiveness measured

### Monitoring & Testing
- [ ] Compliance monitoring program
- [ ] Regular testing/audits
- [ ] Issue tracking and resolution
- [ ] Metrics and reporting
- [ ] Continuous improvement

## Data Privacy Compliance

### GDPR Checklist

**Legal Basis**
- [ ] Legal basis documented for each processing activity
- [ ] Consent mechanism (where applicable)
- [ ] Legitimate interest assessments completed
- [ ] Purpose limitation enforced

**Data Subject Rights**
- [ ] Right to access process
- [ ] Right to erasure process
- [ ] Right to rectification process
- [ ] Right to portability process
- [ ] Response within 30 days

**Data Protection**
- [ ] Data Protection Impact Assessments (DPIAs)
- [ ] Records of Processing Activities (RoPA)
- [ ] Data breach notification procedure
- [ ] Data retention schedule
- [ ] Data minimization practiced

**Organization**
- [ ] DPO appointed (if required)
- [ ] Privacy by design implemented
- [ ] Vendor data processing agreements
- [ ] Cross-border transfer mechanisms
- [ ] Employee training completed

### CCPA/CPRA Checklist

**Consumer Rights**
- [ ] Right to know process
- [ ] Right to delete process
- [ ] Opt-out of sale mechanism
- [ ] "Do Not Sell" link on homepage
- [ ] Non-discrimination policy

**Disclosures**
- [ ] Privacy policy updated
- [ ] Categories of data disclosed
- [ ] Business/commercial purpose stated
- [ ] Third-party sharing disclosed
- [ ] Retention periods stated

**Operations**
- [ ] Consumer request verification
- [ ] Response within 45 days
- [ ] Opt-out preference honored
- [ ] Service provider contracts
- [ ] Employee training

## Compliance Audit Preparation

### Pre-Audit Checklist

**Documentation**
- [ ] Policies current and approved
- [ ] Procedures documented
- [ ] Evidence organized
- [ ] Prior findings addressed
- [ ] Control owners identified

**Access**
- [ ] Auditor access provisioned
- [ ] Interview schedules set
- [ ] System access ready
- [ ] Conference rooms booked
- [ ] Key contacts available

**Evidence Collection**
- [ ] Screenshots gathered
- [ ] Logs exported
- [ ] Reports generated
- [ ] Samples prepared
- [ ] Evidence indexed

### Common Evidence Types

| Control Area | Evidence Examples |
|--------------|-------------------|
| Access Control | User lists, access reviews, termination tickets |
| Change Management | Change tickets, approvals, testing evidence |
| Monitoring | Alert logs, incident tickets, dashboards |
| Training | Completion records, course materials |
| Risk Assessment | Risk register, assessment reports |
| Vendor Management | Contracts, assessments, monitoring |
